        double engagementScore = predictEngagement(user, movie);  
        double satisfactionScore = predictSatisfaction(user, movie);
        
        // 加权融合
        return 0.4 * ctrScore + 0.3 * engagementScore + 0.3 * satisfactionScore;

缓存优化策略

分层缓存架构:

  • L1缓存：用户最近行为 (1小时TTL)
  • L2缓存：用户画像特征 (24小时TTL)
  • L3缓存：全局统计信息 (7天TTL)

预计算策略: 异步预计算活跃用户候选集
